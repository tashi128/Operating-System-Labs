# Operating Systems 2 Lab

This repository contains my **Operating Systems Lab** work implemented in **C**. It includes lab exercises, system-level programming tasks, and practical implementations of fundamental operating system concepts covered during the course.

## Overview

The purpose of this lab is to build practical understanding of how operating systems work internally by implementing core concepts through programming assignments in C.

Topics explored in this repository include:

- process creation and management
- CPU scheduling
- synchronization
- deadlocks
- memory management
- file handling
- inter-process communication
- shell and system call based programs

## Language and Tools

- **Language:** C
- **Compiler:** GCC
- **Platform:** Linux / Unix
- **Version Control:** Git and GitHub
